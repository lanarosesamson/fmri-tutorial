Step 11 — Prepare confounds for FEAT

We now create a smaller confounds file containing only the nuisance regressors we want to include in FEAT.

1

Why create a smaller confounds file?

The original fMRIPrep confounds file contains many columns.

FEAT does not need all of them for this tutorial.

To use a consistent nuisance-regression strategy across all subjects, we create a smaller file containing only the confounds selected for this analysis.

>_

Create the selected confounds file

From the folder containing the confounds file, run: 

python3 - <<'PY'
import pandas as pd

confounds = pd.read_csv(
    "sub-108TUSW011051_ses-1_task-execution_acq-Fs2_run-02_desc-confounds_timeseries.tsv",
    sep="\t"
)

# Six rigid-body motion parameters and their temporal derivatives
base_cols = [
    "trans_x", "trans_y", "trans_z",
    "rot_x", "rot_y", "rot_z",
    "trans_x_derivative1",
    "trans_y_derivative1",
    "trans_z_derivative1",
    "rot_x_derivative1",
    "rot_y_derivative1",
    "rot_z_derivative1"
]

# First 13 anatomical CompCor components
acompcor_cols = [
    f"a_comp_cor_{i:02d}"
    for i in range(13)
]
# Individual regressors for volumes identified as motion outliers
motion_outlier_cols = [
    column
    for column in confounds.columns
    if column.startswith("motion_outlier")
]

selected_cols = base_cols + acompcor_cols + motion_outlier_cols


selected = confounds[selected_cols].fillna(0)

selected.to_csv(
    "confounds.txt",
    sep="\t",
    index=False,
    header=False
)

print("Created confounds.txt")
print(f"Number of volumes: {selected.shape[0]}")
print(f"Number of confound regressors: {selected.shape[1]}")
print(f"aCompCor components: {len(acompcor_cols)}")
print(f"Motion-outlier regressors: {len(motion_outlier_cols)}")
PY

This creates a new file called:

confounds.txt

3

Check the output file

Use head to check the first lines: 

head confounds.txt

The file should contain numerical values only.

It should not contain column names, because FEAT expects a plain text file with regressors.

4

Why remove the header?

The original fMRIPrep file has column names in the first row.

For FEAT, the confound file should contain only numbers.

Each column becomes one nuisance regressor in the model.
