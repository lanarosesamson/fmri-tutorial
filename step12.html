Step 12 — Run FEAT

We now configure the first-level FEAT model for subject 051. We select the BOLD image, add the task EVs, add confounds, define contrasts and launch the analysis.

1

Open FEAT

Open FEAT from the terminal: 

feat

In the FEAT window, choose First-level analysis and Full analysis.

A first-level analysis means that we analyse one subject and one functional run.

Here, we are analysing subject 051, run-02.
2

Data tab

In the Data tab, select the preprocessed functional image and choose where FEAT will save the results.

Output directory

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/sub051_right_left_confounds.feat

FEAT will create a folder ending with .feat. This folder will contain the full first-level analysis.

Input image folder

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fmriprep_output/sub-108TUSW011051/ses-1/func/

Input image

sub-108TUSW011051_ses-1_task-execution_acq-Fs2_run-02_space-MNI152NLin2009cAsym_desc-preproc_bold.nii.gz

This is the preprocessed BOLD image generated by fMRIPrep.

Total volumes 187
TR 2.3 seconds
Delete volumes 0
High pass filter cutoff 100 seconds

Data tab: select the BOLD image, define the output directory and check the number of volumes.

3

Pre-stats tab

In the Pre-stats tab, keep the preprocessing options simple.

The data were already preprocessed by fMRIPrep, but we still keep the FEAT settings used for this tutorial.

Motion correction MCFLIRT
BET brain extraction Enabled
Spatial smoothing 5 mm
Highpass filtering Enabled

Leave the other options unchanged.

Pre-stats tab: smoothing is set to 5 mm and highpass filtering is enabled.

4

Stats tab

In the Stats tab, keep Use FILM prewhitening enabled.

Then add the selected confounds file in the Voxelwise Confound List field.

Confounds file

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/confounds.txt

This file contains nuisance regressors prepared in Step 11. These regressors help control unwanted signal variation.

Task EVs model the experiment. Confounds help clean the model.

Stats tab: select confounds.txt as the voxelwise confound list.

5

Full model setup

Click Full model setup. This opens the General Linear Model window.

Set the number of original EVs to 3, because the experiment has three task conditions.

EV1 Right
EV2 Left
EV3 Both

For each EV, choose Custom (3 column format). This is used because our EV files contain three columns: 

onset   duration   amplitude
Convolution Gamma
Phase 0
Stddev 3
Mean lag 6
Temporal derivative Disabled for this tutorial
Temporal filtering Enabled

We do not add temporal derivatives here, so the design matrix stays simple: one column for Right, one for Left and one for Both.

Full model setup: create three EVs and use custom 3-column files.

6

Add the task EV files

Each EV must point to the correct event file created earlier.

EV1 — Right

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/regressors_right_hand.txt

EV2 — Left

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/regressors_left_hand.txt

EV3 — Both

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/regressors_both_hands.txt

These files tell FEAT when each condition happened during the scan.

EV1: select the file for the right-hand condition.

EV2: select the file for the left-hand condition.

EV3: select the file for the bimanual condition.

7

Contrasts

Go to the Contrasts & F-tests tab.

Set the number of contrasts to 5. These contrasts define the scientific questions asked by the model.

Right > Left 1 -1 0
Left > Right -1 1 0
Right 1 0 0
Left 0 1 0
Both 0 0 1

The three numbers correspond to the three original EVs: Right, Left and Both.

Contrasts: compare Right, Left and Both conditions.

8

Check the design matrix

Click View design before running FEAT.

The design matrix should show three main columns: Right, Left and Both.

If temporal derivatives are disabled, there should not be extra derivative columns.

If the design matrix does not match the model, go back and correct the EV setup before running FEAT.

Design matrix: check that the EVs and contrasts are correct before launching the analysis.

9

Run FEAT

When the Data tab, Pre-stats tab, Stats tab, EVs and contrasts are correct, click Done in the model window.

Then click Go in the main FEAT window.

FEAT will create the output folder:

Expected output folder

/home/lsamson/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/sub051_right_left_confounds.feat

This folder contains the full first-level analysis for subject 051.
