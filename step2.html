Step 2 — Understand fMRI and the BOLD signal

Before analysing the dataset, we need to understand what fMRI measures, what the BOLD signal represents, and what information is contained in a functional image.

1

What is fMRI?

fMRI stands for functional Magnetic Resonance Imaging.

It is a neuroimaging technique used to study changes in brain function over time.

In task fMRI, repeated whole-brain images are acquired while a participant performs an experimental task.

The goal is to identify which brain regions show signal changes associated with the task conditions.

2

What does fMRI actually measure?

fMRI measures changes in the MRI signal over time.

During an fMRI experiment, the scanner repeatedly acquires whole-brain three-dimensional images called volumes.

Each volume represents the entire brain at a specific moment in time.

By comparing successive volumes, we can observe how the MRI signal changes in different parts of the brain throughout the experiment.

fMRI does not measure neuronal activity directly.

These signal changes are linked to the haemodynamic response that follows neural activity.

When neurons become active, they require more oxygen. In response, the brain increases local blood flow and delivers more oxygenated blood to the active region.

The resulting changes in blood oxygenation alter the MRI signal, producing the BOLD signal measured by fMRI.

The fMRI signal is therefore an indirect measure of neural activity rather than a direct recording of neuronal firing.

3

What is inside a task fMRI file?

A task fMRI file is usually a 4D dataset.

The first three dimensions describe space: the x, y and z coordinates of the brain image.

The fourth dimension is time: it corresponds to the succession of volumes acquired during the run.

Each 3D volume is made of many small units called voxels.

For each voxel, the BOLD signal changes across the successive volumes. This produces a voxel-wise time series that can later be modelled statistically.

A task fMRI file contains successive whole-brain volumes. Each voxel has its own BOLD time series.

4

What is the HRF?

HRF stands for Haemodynamic Response Function.

It describes the typical shape of the BOLD signal following a brief burst of neural activity.

Because the BOLD response is delayed and spread over several seconds, FEAT models the expected signal using the HRF rather than the task timing alone.

5

What is a task fMRI analysis trying to do?

The aim of task fMRI analysis is to test whether variations in the BOLD signal are associated with specific periods of the experiment.

In practice, we compare the measured voxel time series with a predicted response based on the task model.

If the measured signal matches the predicted response well, this suggests that the voxel is related to the experimental condition.

Task timing is transformed into an expected BOLD response using the HRF, then compared with the measured voxel signal.

In our case, we want to model the signal changes associated with three motor conditions:

  • right-hand movements
  • left-hand movements
  • both-hand movements

Our main interest is the comparison between Right and Left movements.

The Both condition is also included so that we can practise building a complete first-level (subject-level) FEAT model using multiple experimental conditions.
