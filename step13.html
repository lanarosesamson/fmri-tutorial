Step 13 — View FEAT outputs

The first-level FEAT analysis is complete. In this step, we check that FEAT worked, locate the statistical maps and open them in FSLeyes.

1

Check that FEAT worked

Before opening the results, first check that FEAT created the output folder. 

ls ~/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/

If FEAT worked, you should see: 

sub051_right_left_confounds.feat

Then check that the statistical maps were created: 

ls ~/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/sub051_right_left_confounds.feat/stats/

You should see files such as: 

cope1.nii.gz
cope2.nii.gz
cope3.nii.gz
cope4.nii.gz
cope5.nii.gz

zstat1.nii.gz
zstat2.nii.gz
zstat3.nii.gz
zstat4.nii.gz
zstat5.nii.gz
If these files exist, the FEAT analysis completed successfully.
2

Open the maps in FSLeyes

Open FSLeyes: 

fsleyes

Then use File → Add from file.

First load:

  • MNI152_T1_2mm_brain.nii.gz as anatomical background
Path:/fsl/data/standard/MNI152_T1_2mm_brain.nii.gz
The anatomical image is the background. The zstat files are displayed as coloured overlays.
3

Find the FEAT output folder

The FEAT output folder is located here:

FEAT output folder

~/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/sub051_right_left_confounds.feat/

Inside this folder, the most important folder for visualisation is stats/.

Stats folder

~/fMRI_tutorial_project/righthanded/bids/derivatives/fsl_output/sub_51/sub051_right_left_confounds.feat/stats/

Navigate to the FEAT output folder, then open the stats folder.

3

Understand the zstat files

Each zstat file corresponds to one contrast defined in FEAT.

zstat1.nii.gz Right > Left
zstat2.nii.gz Left > Right
zstat3.nii.gz Right > Rest
zstat4.nii.gz Left > Rest
zstat5.nii.gz Both > Rest

The numbering follows the order of the contrasts created in Step 12.

4

Example: zstat1 and zstat2

The example below shows two contrasts displayed at the same time.

Blue overlay zstat1 = Right > Left
Red overlay zstat2 = Left > Right

In this example, blue regions are more active during right-hand movements than left-hand movements.

Red regions are more active during left-hand movements than right-hand movements.

Because these two contrasts test opposite comparisons, activations can appear in opposite hemispheres.

Example in FSLeyes: zstat1 is displayed in blue and zstat2 in red.

The colours are only display choices in FSLeyes. They do not change the statistical results.
