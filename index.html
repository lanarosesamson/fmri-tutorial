Step-by-step fMRI Analysis tutorial

A step-by-step guide to analysing a motor fMRI dataset with FSL.

This website is a practical tutorial for beginners.

It explains how to analyse a simple motor fMRI experiment using FSL.

The tutorial follows a real pipeline, from one subject to a group analysis.

Start the tutorial → View all steps

What is this tutorial about?

We will analyse a motor fMRI task based on hand movements.

  • understand the experiment
  • understand fMRI and the BOLD signal
  • work with BIDS datasets
  • run fMRIPrep
  • create EV files
  • prepare confounds for FEAT
  • run a first-level FEAT analysis
  • view the results in FSLeyes
  • automate the analysis pipeline
  • perform a group analysis
  • interpret brain activations

What do you need before starting?

  • Ubuntu access
  • FSL installed
